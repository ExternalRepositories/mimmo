Detailed Description

Structured lattice.

Date
24/mar/2016
Authors
Rocco Arpa
Edoardo Lombardi

Basically, it builds an elemental 3D shape (box, sphere, cylinder or part of them) around the geometry and set a structured cartesian mesh of control points on it (lattice). NO displacements for control points and NO NURBS parameters for FFD are present in this structure, only geometrical information are stored in the object.

WARNING AbsorbSectionXML, FlushSectionXML not coded! It's using baseManipulation default;

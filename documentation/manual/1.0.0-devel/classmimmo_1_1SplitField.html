Detailed Description

SplitField is an abstract executable block class capable of splitting fields associated to a stitched MimmoObject, once the original geometries of the stitched object and its division maps are set.

Date
30/nov/2016
Authors
Rocco Arpa

SplitField is an abstract class. To use its features take a look to its specializations, here presented as derived class, SplitScalarField and SplitVectorField.

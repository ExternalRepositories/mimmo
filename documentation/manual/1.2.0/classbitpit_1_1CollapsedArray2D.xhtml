Detailed Description

template<class T>
class bitpit::CollapsedArray2D< T >

Metafunction for generation of a collapsed array of arrays.

Usage: Use CollapsedArray2D<Type> to declare a collapsed array of arrays.

Template Parameters
TThe type of the objects stored in the array

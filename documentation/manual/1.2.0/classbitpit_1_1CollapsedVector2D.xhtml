Detailed Description

template<class T>
class bitpit::CollapsedVector2D< T >

Metafunction for generation of a collapsed vector of arrays.

Usage: Use CollapsedVector2D<Type> to declare a collapsed vector of arrays.

Template Parameters
TThe type of the objects stored in the vector

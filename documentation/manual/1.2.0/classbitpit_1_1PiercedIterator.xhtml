template<typename value_t, typename id_t = long, typename value_no_cv_t = typename std::remove_cv<value_t>::type>
class bitpit::PiercedIterator< value_t, id_t, value_no_cv_t >

Iterator for the class PiercedVector.

Usage: use PiercedVector<value_t, id_t>::iterator to declare an iterator for a pierced vector, use PiercedVector<vaue_t, id_t>::const_iterator to declare a const iterator for a pierced vector.

Template Parameters
value_tThe type of the elements stored in the vector
id_tThe type of the ids to associate to the elements

Detailed Description

Executable block that provides the computation of a 3D array field over a 3D mesh. The field is calculated solving a Laplacian problem over the mesh with given boundary conditions.

Prescribed (Dirichlet) field boundary conditions are explicitly provided by the User, identifying boundaries through MimmoObject patches and associating to them the value of each component of the field as Dirichlet condition. Optionally an inpermeability-like/slip condition (zero vector field normal to boundary surface) can be imposed on chosen boundary patches.

The block can perform multistep evaluation to relax field propagation

Class/BaseManipulation Object specialization of class PropagateField for the propagation in a volume mesh of a 3D array field.

The xml available parameters, sections and subsections are the following :

Inherited from BaseManipulation:

  • ClassName : name of the class as mimmo.PropagateVectorField;
  • Priority : uint marking priority in multi-chain execution;
  • Apply : if set to 1, apply propagated field to target geometry as a deformation field;
  • PlotInExecution : plot optional results in execution;
  • OutputPlot : path to store optional results.

Inherited from PropagateField:

  • Solver : 1-true use direct Laplacian solver, 0-false use iterative Smoother (default 0);
  • WeightConstant : coefficient used to get weights of the stencil points in function of distance (1.0 default);
  • Dumping : 1-true activate dumping control, 0-false deactivate it.
  • DumpingType : 0- distance control, 1-volume control.
  • DumpingInnerDistance : inner limit of dumping function eta, if dumping is active;
  • DumpingOuterDistance : outer limit of dumping function eta, if dumping is active;
  • DecayFactor : exponent to modulate dumping function (as power of), if dumping is active
  • SmoothingSteps : number of steps the Smoother solver need to perform (1 default);
  • Convergence : convergence flag for smoothing solver;
  • Tolerance : convergence tolerance for laplacian smoothing and direct solver;

Proper fo the class:

  • MultiStep : got deformation in a finite number of substep of solution;
  • SlipNormalRatio : value > 1, meant to adjust defects in vertex-normals of candidate slipsurface. if the rate between the maximum component of the normal and the candidate component a is greater then this value, the candidate component will be set to 0, and the normal will be recalculated.

Geometry, boundary surfaces, boundary condition values for the target geometry have to be mandatorily passed through ports.

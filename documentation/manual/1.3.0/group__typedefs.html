◆ setBackgroundDetails()

void mimmo::ControlDeformExtSurface::setBackgroundDetails ( int  nCell = 50)

Set the number of Cells NC necessary to determine the spacing of a background grids. A background axis-aligned cartesian volume grid wrapping each constraint + deformed body is created to evaluate distances of each deformed point from constraint surface. Spacing of such grid is calculated as dh = D/NC where dh is the spacing, D is the diagonal of grid bounding-box.

Parameters
[in]nCellnumber of cells for cartesian background grid
